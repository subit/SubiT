* 	In order to use & develop the source, Python 2.7.x is required:
	http://www.python.org/download/releases/

* 	Additionally, in order to use the GUI, PySide (Qt) is required:
	http://qt-project.org/wiki/PySideDownloads

* 	And Finally, in order to create an executable version of SubiT, PyInstaller 
	is needed:
	http://www.pyinstaller.org

*	SubiT was developed using "Python Tools for Visual Studio": 
	http://pytools.codeplex.com/

* 	In order to create the installation file of SubiT, "Inno Setup" is required:
	http://www.jrsoftware.org/isinfo.php

**	Building the ContextMenu handler for windows, requires Visual Studio 2010.
